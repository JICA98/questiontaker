# QuestionTaker - AIIMS CRE Practice App

QuestionTaker is an Android application designed to help medical professionals and students prepare for the AIIMS CRE (Common Recruitment Examination), with a special focus on Anesthesia and Operation Theatre (OT) Technician roles.

## Features

*   **Comprehensive Question Bank**: Access hundreds of practice questions covering Capnography, Airway Devices, Medical Gases, Pharmacology, OT Environment, Sterilization, and more.
*   **Dynamic Tagging**: Each practice paper features detailed 3-4 specific topic tags to quickly identify the paper's contents.
*   **Offline-First & Cloud Sync**: The app instantly loads using local assets. When connected to the internet, it seamlessly syncs the latest question bank directly from GitHub in the background.
*   **Mock Exams**: Take timed mock exams with customizable question counts and durations.
*   **Starred Review Room**: Bookmark difficult questions during practice or exams and review them later in a dedicated room.
*   **Premium Material 3 Design**: Enjoy a beautiful, smooth, and edge-to-edge UI experience on modern Android devices.

## Installation

You can download the latest version of QuestionTaker directly from the GitHub Releases page:

1. Go to the [Releases](../../releases) section of this repository.
2. Download the latest `app-release.apk` file.
3. Open the downloaded file on your Android device and install it (you may need to allow installations from unknown sources).

## Building from Source

To build the app yourself:

1. Clone the repository.
2. Open the project in Android Studio.
3. Generate a keystore or build a debug version.
4. Run `./gradlew assembleDebug` or `./gradlew assembleRelease`.
